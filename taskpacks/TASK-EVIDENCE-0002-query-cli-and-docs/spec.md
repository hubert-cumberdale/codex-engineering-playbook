# Spec: Evidence Index Query CLI and Documentation

## Objective
Add a deterministic, read-only CLI to query the evidence index generated by Task Pack A, and document its usage in Tier-2 docs.

## Requirements
- Extend `tools.evidence.cli` with query subcommands:
  - `list-runs`
  - `list-artifacts --run-id <RUN_ID>`
  - `show-artifact --run-id <RUN_ID> --type <TYPE>`
  - Optional: `show-index-meta`
- Query commands must read from the index JSON only; no filesystem scanning.
- Deterministic, stable output and exit codes for user errors.
- Add tests covering each command and error cases (missing index file, invalid JSON, unknown run id, ambiguity handling).
- Add Tier-2 documentation at `docs/evidence/EVIDENCE_INDEX.md`.
- Add a short workflow pointer in `docs/CHAT_PLAYBOOK.md`.

## Constraints
- No network access.
- No semantic inference or heuristic behavior.
- No new dependencies.
- No changes to orchestrator runtime behavior.
